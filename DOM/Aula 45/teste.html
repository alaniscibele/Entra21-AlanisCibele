Exercícios 1) Crie uma classe Ponto2D para representar um ponto no espaço cartesiano. Essa classe deve ter os atributos: x e y. Essa classe deve ter o método estático temEixoComum(pontoA, pontoB) que retorna true se as duas classes tiverem as mesmas coordenadas para algum dos eixos. Exemplo: (1, 2) e (1, -17) -> True (13, -8) e (8, -22) -> False
